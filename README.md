# API

Blogs Routes:

GET /blogs
GET /blogs/:blogId  
POST /blogs
PUT /blogs/:blogId
DELETE /blogs/:blogId
PATCH /blogs/:blogId/toggle-publish
GET /myBlogs

Comments Routes:
GET /blogs/:blogId/comments
POST /blogs/:blogId/comments
GET /blogs/:blogId/comments/:commentId
PUT /blogs/:blogId/comments/:commentId
DELETE /blogs/:blogId/comments/:commentId
